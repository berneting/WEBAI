# WEBAI
## [writing_01](https://berneting.github.io/WEBAI/writing_01.html) 文學寫作提示詞生成器（文學風格）
將原始文章、摘要、段落或主題貼入上方，透過文體、情緒、敘事視角、意象密度與句子張力等設定， 自動生成更具文學層次與改寫方向的 AI 提示詞。
文學各類風格選配

## [writing_02](https://berneting.github.io/WEBAI/writing_02.html) 行銷用文宣生成器（商用術語）
使用方式：
1. 輸入要推廣的產品或服務，例如：海產、文具、老齡健康顧問、課程。
2. 選擇文案風格。
3. 點擊「生成內容」。
4. 上方可參考文宣句，下方可直接複製「AI 提示句」貼回 ChatGPT 或其他 AI 工具。
5. 員工不需要自行想提示語，只要複製即可。

## [writing_03](https://berneting.github.io/WEBAI/writing_03.html) 研究摘要轉社媒推廣提示句生成器（短文）
把研究摘要與研究後討論分開輸入，讓 AI 不只是「安全改寫」，而是能抓出真正值得被看見的核心觀點， 寫出更有主軸、更有溫度、也更能讓一般讀者讀下去的研究推廣文字。

## [writing_04](https://berneting.github.io/WEBAI/writing_04.html) 摘要轉 AI 提示句生成器（中長文）
把文章摘要、研究摘要、新聞導言或課堂整理貼進來，選擇結構、語氣與寫作模式，就能生成一段可直接貼回 AI 的高精度提示句。
這個版本已加入「避免只像在評論摘要」的限制設計，讓產出的文章更接近獨立社論、雜誌專欄或真正的公共評論文章。
